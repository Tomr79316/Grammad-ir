An Gramadóir:
Web Interface

An leathanach seo as Gaeilge.

There is also a version of An Gramadóir integrated into Microsoft Word, and available for purchase from Cruinneog.

  • Type (or cut and paste) some Irish text in the box below (at most 1024 characters).
  • You can view the error messages in any of the languages listed below by selecting one of the buttons (you can't check the grammar of these languages; Irish only at the moment).
  • Click the "Send" button.
