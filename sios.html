When this package becomes available as part of Linux distributions, or is integrated into word processing software, I will put the appropriate links here. I will continue to support the standalone version however, since I am old school and like the command line interface.

I am hoping that users will develop more appealing front-ends for An Gramadóir. In particular, it should be easy to construct a simple web form interface which allows text to be submitted and which serves back the HTML generated by "gr --html".

I am also looking for developers who might be interested in helping to integrate An Gramadóir into some standard text editors. Emacs might be a good place to start.
