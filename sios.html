An Gramadóir:
Download/Installation/Usage

Users

If you just want to install and use the Irish grammar checker, you probably don't need to download anything from this site; instead you should install the Perl module Lingua::GA::Gramadoir as described below.

Developers

All developers will need the latest version of the developers' pack, which contains the scripts for converting rule sets into an installable Perl module.

All of the language-dependent input files used by the developers' pack (lexicons, rule sets, etc.) are contained in the following language packs:

  • gramadoir-ga (Irish)

Installation and Usage

The Irish grammar checker is distributed as the Perl module Lingua::GA::Gramadoir. If you are using Linux or some other Unix-like operating system, you surely already have Perl installed and you can install the module using CPAN.pm. If you are running something primitive like Windows, you'll first have to install ActiveState Perl and then install An Gramadóir using ppm.

You can also install manually: download, unpack the archive, and try the following standard procedure: 

% perl Makefile.PL
% make
% make install

You should then have the front-end script gram-ga.pl installed. To check the grammar of an Irish language text file called gaeilge.txt, use the following from the command line: 

% gram-ga.pl gaeilge.txt

More detailed usage instructions can be found using: 

% gram-ga.pl --help
