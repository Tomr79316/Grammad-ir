An Gramadóir

Kevin P. Scannell

Summary

This is the home page for An Gramadóir, the first grammar and style checker for the Irish language. In fact, to the best of my knowledge, this is the first such package for any minority language.

You are welcome to download the software for free from this site. It is released under the GNU General Public License which means (roughly speaking) that you are free to copy, modify, or even sell this software as long as redistributed versions preserve these same freedoms. Please consider contributing your own rules to help improve later releases.

System Requirements

This is software for Unix and Linux platforms. If you are an Irish speaker and are using, say, Windows, you should format your hard drive and install something like Gentoo Linux. It's free and the grassroots translation of GNU/Linux into Irish is well underway. Otherwise, have fun waiting for Microsoft to release Windows in Irish.

There are (intentionally) very few software dependencies for this package. Everything you need should come pre-loaded with any self-respecting Linux distribution: bash, sed, awk, perl, grep, and a C compiler. Users interested in contributing rules will probably also want flex and bison. I have managed to build and test everything under Gentoo Linux, Solaris 8, and on a DEC alpha. There is no a priori reason why you shouldn't be able to get things working with Mac OS X if you have built the programs above, but I haven't tried it yet.

Installation and Usage

First, Download the tarball. Then (assuming you want to install to /usr/local and you have the appropriate permissions) try the following standard installation process: 

% gunzip gramadoir-x.x.tar.gz
% tar xvf gramadoir-x.x.tar
% cd gramadoir-x.x
% ./configure
% make
% make install
% make installcheck

The front-end script is called "gr". To check the grammar of an Irish language text file called gaeilge.txt, use the following: 

% gr gaeilge.txt

Each sentence containing an error is written to standard output. The sentence is preceded by the line number (in the original text file) on which it starts and is followed by a brief description of the errors found in the sentence. You can see some actual output on the Sampler page, or enter some of your own text into the Web Interface.

If there are words you would like to be disregarded while checking spelling and grammar you can create a plain text file in your home directory called ".neamhshuim". The first line should contain the number of words in the file and then place one word on each line.

There are also a few command line options:

  • --litriu: Write all of the unknown/misspelled words to standard output.
  • --html: Produce HTML output which should resemble the output of vanilla "gr" when viewed in a web browser.
  • --xml: Write the intermediate tagged XML stream to standard output; useful for debugging.
  • --iomlan: Report all errors (do not use the file "~/.neamhshuim").
  • --teanga=XX: Specify the two-letter ISO-639-1 code for the language of the text to be checked. Default is ga (Irish).
  • --ionchod=ENC: Specify the character encoding of the input text. The default depends on the language; Irish uses ISO88591.
  • --help: Display some help and exit.
  • --version: Display version and copyright information and exit.
