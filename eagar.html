Emacs

Martin Gregory has kindly contributed an Emacs interface to An Gramadóir which is very easy to use. It shows you a split screen with the original text file on top and the errors output by An Gramadóir below. With a single keystroke, you can move through the errors in parallel, making corrections as you go. Here are his instructions:

"If you are an experienced Emacs user, you'll probably want to auto load the file and define your own keys for the two functions listed below.

  1. put gramadoir.el in your emacs site lisp directory, if you don't have one, put it it ~/lisp
  2. put the following lines in your ~/.emacs file 
       (load-file "~/lisp/gramadoir.el")
   (global-set-key "\C-cg" 'gramadoir-check-files)
   (global-set-key [f3] 'gramadoir-next-message)
  3. start Emacs
  4. To have An Gramadóir check files, press the key ^C followed by the key g.
  5. You are prompted for files to check, enter them using either relative or absolute paths, but avoid using ~ or environment variables as gramadoir.el doesn't handle these yet.
  6. A second window will open with the messages from gr.
  7. To go to the first error position in the first file, press F3. Repeated pressing of F3 takes you to each subsequent error."
